# Tomorrow Desk 使用方法

Tomorrow Desk 是一个用于记录“明天开机后第一眼要看到什么”的桌面待办小工具。它会把你输入的内容保存在本机，并在开启开机启动后，于下次登录 Windows 时自动显示窗口。

## 安装与启动

1. 双击安装包安装：
   `dist/Tomorrow Desk Setup 0.1.0.exe`
2. 安装完成后打开 `Tomorrow Desk`。
3. 第一次打开后，建议确认右上角 `Login` 和 `Top` 都处于金色高亮状态。

如果不想安装，也可以运行便携版：
`dist/Tomorrow Desk 0.1.0.exe`

安装版的开机启动更稳定，建议日常使用安装版。

## 基本输入

- 在主输入框中输入待完成事项。
- 按 `Enter`：把当前输入固定为一条待完成事项，并显示在输入框下方列表中。
- 按 `Shift + Enter`：在输入框内换行，不会固定为事项。
- 输入内容会自动保存。
- 如果还没按 `Enter`，当前输入会作为草稿保存，下次打开仍会恢复。

## 窗口按键

### Top

切换窗口置顶。

- 金色高亮：窗口保持在其他窗口上方。
- 非高亮：窗口按普通窗口显示。

建议保持开启，这样开机后窗口更容易被第一眼看到。

### Login

切换开机启动。

- 金色高亮：登录 Windows 后自动启动 Tomorrow Desk。
- 非高亮：不会自动启动，需要手动打开。

要实现“开机后立刻看到待办事项”，请保持 `Login` 开启。

### _

最小化窗口到任务栏。

### x

隐藏到系统托盘，不退出程序。

关闭前会先保存当前输入。隐藏后可以从系统托盘图标重新打开。

### 清空

清空当前输入框和下方待完成事项列表。

点击后会直接把输入光标放回输入框，可以马上继续输入新的事项。这个按钮不再归档内容；它只是清空当前页面。

## 托盘菜单

程序隐藏后，可以在 Windows 系统托盘中找到 Tomorrow Desk 图标。

右键托盘图标会看到：

- `Show`：重新显示窗口。
- `Archive`：归档当前内容并清空页面。
- `Quit`：保存后退出程序。

日常只想把窗口收起来时，建议点窗口右上角 `x`，不要用 `Quit`。如果退出了程序，它就不会继续停留在托盘中。

## 如何让开机后立刻看到待办事项

推荐流程：

1. 使用安装包安装 Tomorrow Desk。
2. 打开程序。
3. 确认右上角 `Login` 是金色高亮。
4. 确认右上角 `Top` 是金色高亮。
5. 睡前在输入框里写明天要做的事。
6. 可以按 `Enter` 把事项固定到列表，也可以保留为草稿。
7. 关机或重启电脑。
8. 第二天登录 Windows 后，Tomorrow Desk 会自动启动并显示窗口。

如果第二天没有自动显示：

1. 手动打开 Tomorrow Desk。
2. 再次确认 `Login` 是金色高亮。
3. 优先使用安装版，而不是便携版。
4. 检查 Windows 设置中的启动应用，确认 Tomorrow Desk 没有被系统禁用。

## 数据保存说明

- 待完成事项和草稿保存在本机，不上传到网络。
- 输入会自动保存；隐藏窗口、退出程序、关机前也会尝试保存。
- `清空` 会清空当前页面并保存为空。
- 托盘菜单中的 `Archive` 会把当前内容归档后再清空。

## 建议使用习惯

睡前只做三件事：

1. 打开 Tomorrow Desk。
2. 写下明天第一件要处理的事情。
3. 确认 `Login` 和 `Top` 是开启状态。

第二天开机后，就让它把昨天晚上的“交接棒”递回给你。
